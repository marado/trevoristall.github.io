Cakes by Jane

  • Design: The Wired Mouse
  • Code: Trevor Allen (Me)
  • Project Head(s) The Wired Mouse

Cakes by Jane is a local gourmet cake shop here in Asheville. I built this website on the Wordpress platform with woocommerce since one of the unique requirements for the project was the ability to ship each item in an order to a seperate address. There is a lot of custom Javascript and CSS animation, with browser support back to IE9.
